# 検出能力向上ガイド（クイックリファレンス）

## 1. まず試すべき設定（無料・すぐできる）

### 標準設定
```
メソッド: relative
相対閾値: 0.15
エッジ強調: OFF
Select Largest Particle: ON
```

### 夜間の誤検出が多い場合
```
相対閾値: 0.25 に上げる
ROI: 設定する（個体がいる範囲のみ）
Temporal Median Filter: ON
```

### コントラストが低い場合
```
エッジ強調: ON（重み 0.3）
CLAHE: ON（制限 2.5）
```

## 2. エッジ強調の使い方（新機能）

### いつ使う？
- 個体の輪郭がぼやけている
- 背景とのコントラストが低い
- 照明が弱い

### 設定
```
エッジ強調: チェック
重み: 0.2 ～ 0.5
  - 0.2: 軽い強調（夜間推奨）
  - 0.3: 標準（推奨）
  - 0.5: 強い強調（ノイズに注意）
```

### 注意点
- 重みを大きくしすぎるとノイズも強調される
- 夜間は0.2～0.3程度に抑える
- プレビューで効果を確認しながら調整

## 3. パラメータ調整の順序

1. **相対閾値を調整**（最も重要）
   - 誤検出が多い → 上げる（0.20～0.30）
   - 見逃しが多い → 下げる（0.10～0.15）

2. **ROIを設定**
   - 個体がいる範囲のみに制限
   - 誤検出が大幅に削減される

3. **エッジ強調を試す**
   - コントラストが低い場合に有効
   - 重み0.3から開始

4. **Temporal Median Filterを使用**
   - 背景ノイズが多い場合
   - フレーム数100で開始

## 4. 画像取得環境の改善（低コスト・高効果）

### すぐできる改善
- 背景を白無地に変更
- 照明を斜め45度から当てる
- カメラの自動調整をOFFにして固定

### 投資が必要だが効果大
- 赤外線照明の導入（昼夜で安定）
- マット仕上げの背景板
- 拡散照明システム

## 5. トラブルシューティング

| 症状 | 原因 | 対策 |
|------|------|------|
| 夜間誤検出多い | 照明暗い | 相対閾値0.25、ROI設定 |
| 検出されない | コントラスト低 | エッジ強調ON、閾値下げる |
| 不安定 | 照明変動 | カメラ設定固定、赤外線照明 |
| ノイズ多い | 画質悪い | ガウシアンブラー、モルフォロジー |

## 7. 推奨改善手順

### ステップ1: パラメータ最適化（1時間）
1. 相対閾値を0.20に設定
2. ROIを設定
3. エッジ強調を試す（重み0.3）
4. プレビューで確認

### ステップ2: 簡単な環境改善（1日）
1. 背景を白無地に変更
2. 照明の位置を調整
3. カメラ設定を固定

### ステップ3: 本格的な改善（1週間）
1. 赤外線照明を購入・設置
2. 専用容器を作成
3. 拡散照明システムを構築

## 8. よくある質問

**Q: エッジ強調は常にONにすべき？**
A: 必要な場合のみ。照明が良好ならOFFで十分。

**Q: 一番効果的な改善は？**
A: 照明の改善（赤外線照明）＋ROI設定の組み合わせ。

**Q: 画像処理だけで十分？**
A: 画像処理には限界あり。環境改善が最も効果的。

**Q: 夜間の誤検出が減らない**
A: 相対閾値を0.30まで上げる＋ROI設定＋Temporal Median Filter。
   それでもダメなら照明改善が必要。
